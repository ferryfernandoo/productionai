💻 HTML Editor Feature

Complete Documentation & Implementation Guide

📋 Feature Overview

Memungkinkan user untuk melihat, mengedit, dan mengunduh HTML code yang dihasilkan oleh bot dalam format file .html.

🎯 User Workflow

Step Action Description
1 Request HTML User meminta bot untuk membuat HTML page
2 Bot Response Bot mengirim pesan dengan HTML code
3 Show Button Button "💻 HTML" muncul di bawah message
4 Open Editor User klik tombol untuk buka HTML editor modal
5 Edit & Preview User edit HTML dan preview di tab baru
6 Download User klik download untuk simpan file .html

HTML Detection Patterns

  • <html[^>]*>[\s\S]*</html>
  • <body[^>]*>[\s\S]*</body>
  • <div[^>]*>[\s\S]*</div>
  • <!DOCTYPE[^>]*>[\s\S]*</html>
