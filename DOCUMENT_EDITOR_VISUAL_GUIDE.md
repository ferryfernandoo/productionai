ChatBot Interface

```
Document Editor opens
│

│  ├─ 📄 DOCX  (Word documents)
│  ├─ 📊 PPTX  (Presentations)
│  └─ 📈 EXCEL (Spreadsheets)

```

├─ Click "🤖 AI" button (top right)
├─ Type your request: "Buatkan artikel tentang AI"
├─ Click "✨ Generate & Insert"
└─ Content appears in document!

## 📚 Editor Type Guides

### DOCX (Word Document)

**Layout:**
```
[Document Title] [DOCX] [PPTX] [EXCEL] [🤖 AI] [⬇️ Export]
┌─────────────────────────────────────────────────────┐
│                                                     │
│  Paragraph 1 - Click to edit                       │
│  Paragraph 2 - Another paragraph                   │
│  Paragraph 3 - More content                        │
│                                                     │
│                        ┌─────────────────┐         │
│                        │ 🤖 AI Assistant │         │
│                        │                 │         │
│                        │ Prompt:         │         │
│                        │ [text box]      │         │
│                        │                 │         │
│                        │ [Generate]      │         │
│                        │                 │         │
│                        │ Templates:      │         │
│                        │ 📝 Article      │         │
│                        │ 📋 Summary      │         │
│                        └─────────────────┘         │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**Workflow:**
```
1. Type title: "Artikel tentang Machine Learning"
2. Click 🤖 AI
3. Type: "Buatkan 3 paragraf tentang ML"
4. Click Generate
5. Content auto-inserted below title
6. Edit if needed
7. Click Export → Download .docx file
```

### PPTX (Presentation)

**Layout:**
```
Slide 1:
┌───────────────────────────┐
│   Judul Slide Pertama     │  ← Editable
├───────────────────────────┤
│   Konten slide di sini    │  ← Editable
│                           │
│   - Point 1               │
│   - Point 2               │
└───────────────────────────┘

Slide 2:
┌───────────────────────────┐
│   Slide Kedua             │
├───────────────────────────┤
│   Lebih banyak konten     │
└───────────────────────────┘

Notes for Slide: [Presenter notes here]
```

**Workflow:**
```
1. Click 🤖 AI
2. Type: "Buatkan 5 slide presentasi tentang AI"
3. Click Generate
4. AI creates slides automatically
5. Edit title and content per slide
6. Click Export → Download .pptx file
```

### EXCEL (Spreadsheet)

**Layout:**
```
┌─────┬──────────┬──────────┬──────────┐
│ ID  │ Product  │ Quantity │ Price    │
├─────┼──────────┼──────────┼──────────┤
│ 1   │ Item A   │ 10       │ 5000     │
├─────┼──────────┼──────────┼──────────┤
│ 2   │ Item B   │ 20       │ 3000     │
├─────┼──────────┼──────────┼──────────┤
│ 3   │ Item C   │ 15       │ 4000     │
└─────┴──────────┴──────────┴──────────┘
```

**Workflow:**
```
1. Click cell to edit
2. Type value
3. Press Tab to move to next cell
4. Click 🤖 AI for formula/data generation
5. Type: "Generate sales data for Q1"
6. Click Export → Download .csv file
```

## ❌ Troubleshooting

### Problem: Export tidak berhasil
```
Solutions:
1. Check file size (biasanya OK)
2. Check browser permissions
3. Try different browser
4. Check internet connection
5. Refresh page and try again

If still stuck:
- Export di browser lain
- Copy content manually
- Contact support
```

### Problem: AI generation timeout
```
Solutions:
1. Check internet speed
2. Try simpler prompt
3. Use quick template instead
4. Wait a few seconds before retry
5. Try smaller request (3 slides instead of 10)
```

### Problem: Document tidak tersimpan
```
Solutions:
1. Auto-save berjalan setiap 10 detik (normal!)
2. Check browser console (F12)
3. Try clearing cache
4. Export document as backup
5. Refresh page

Note: Uses localStorage, not cloud storage (yet)
```

### Problem: Export file corrupted
```
Solutions:
1. Try export lagi
2. Use different browser
3. Check antivirus (might quarantine)
4. Manual copy-paste content
5. Report if persists
```
