# 🎨 Watermark Feature Setup

## Overview
Images generated by AI now automatically include "DEEPERNOVA" watermark in the bottom-right corner with 50% transparency.

## Installation

### 1. Install Sharp (Image Processing Library)

Navigate to server folder and install sharp:

```bash
cd server
npm install sharp
```

Or use yarn:
```bash
cd server
yarn add sharp
```

### 2. Verify Installation

Check that sharp is installed:
```bash
npm ls sharp
```

You should see: `sharp@<version>`

## How It Works

### Image Generation Flow

1. **Image Generated**: TokenMix API generates image
2. **Watermark Applied**: Sharp library downloads image and adds "DEEPERNOVA" text
   - Position: Bottom-right corner
   - Size: Scales with image (larger images = larger text)
   - Opacity: 50% transparency (opacity: 0.5)
   - Color: White
3. **Saved Locally**: Watermarked image saved to `public/` folder
4. **Returned**: Public URL returned to frontend instead of original TokenMix URL

### File Structure

```
public/
  watermarked-<uuid>.png  ← Generated watermarked images stored here
```

## Configuration

### Watermark Text
To change watermark text from "DEEPERNOVA" to something else, edit:

**File**: `server/server.js`
**Function**: `addWatermarkToImage(imageUrl, watermarkText)`
**Usage**: `addWatermarkToImage(imageUrl, 'YOUR_TEXT')`

Current line (~2350):
```javascript
const watermarkedBuffer = await addWatermarkToImage(imageUrl, 'DEEPERNOVA');
```

### Watermark Styling

Position: Bottom-right (`text-anchor: end`, `dominant-baseline: text-bottom`)
Padding: 20px from edges
Font: Arial Bold
Opacity: 50% (0.5)

To modify, edit the SVG in `addWatermarkToImage()` function:

```javascript
const watermarkSvg = Buffer.from(`
  <svg width="${width}" height="${height}" xmlns="http://www.w3.org/2000/svg">
    <text 
      x="${x}" 
      y="${y}" 
      font-family="Arial, sans-serif" 
      font-size="${fontSize}" 
      font-weight="bold" 
      fill="white" 
      opacity="0.5"    <!-- Change this to 0.3 for less visible, 0.7 for more visible -->
      ...
    >${watermarkText}</text>
  </svg>
`);
```

## Troubleshooting

### Error: "Cannot find module 'sharp'"

**Solution**: Install sharp again
```bash
cd server
npm install sharp
```

### Watermark Not Appearing

**Checks**:
1. ✅ Is sharp installed? (`npm ls sharp`)
2. ✅ Do public folder exist? (`ls public/`)
3. ✅ Check server logs for watermark errors: `[WATERMARK]` prefix

### Watermark Looks Blurry or Distorted

This might happen if:
- Image size is very small (< 100px width)
- Font scaling is too aggressive

**Solution**: Adjust min font size in `addWatermarkToImage()`:
```javascript
const fontSize = Math.max(40, Math.floor(width / 20)); // Change 40 to different value
```

## Performance Notes

- Sharp library adds ~100-500ms to image generation (depending on image size)
- Watermarked images stored locally (public folder)
- Images will grow by ~5-10% in file size due to watermark text
