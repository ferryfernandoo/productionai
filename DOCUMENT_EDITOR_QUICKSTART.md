# AI Document Editor - Quick Start Guide

## ⚡ Getting Started in 2 Minutes

### Step 1: Access Document Editor
1. Open Deepernova AI Chat
2. Click the `+` button in the bottom left
3. Select `📝 Doc Editor`

### Step 2: Choose Document Type
- **📄 DOCX** - Write articles, reports, letters
- **📊 PPTX** - Create presentations
- **📈 EXCEL** - Build spreadsheets

### Step 3: Start Creating
- **Type directly** into the editor
- **Or use AI** - Click `🤖 AI` button to generate content
- **Export** - Click `⬇️ Export` to download your file

---

## 🎯 Common Workflows

### Creating an Article with AI

```
1. Select DOCX format
2. Click "🤖 AI" on top right
3. Type: "Buatkan artikel tentang machine learning dengan 5 paragraf"
4. Click "✨ Generate & Insert"
5. Content akan ditambahkan ke dokumen
6. Edit jika perlu
7. Click "⬇️ Export" untuk download
```

### Creating a Presentation

```
1. Select PPTX format
2. Use Quick Template "Article" or type slide content directly
3. Use AI to generate slide content
4. Each slide has Title and Content sections
5. Export untuk download
```

### Creating a Spreadsheet

```
1. Select EXCEL format
2. Click pada cell untuk edit
3. Struktur data dalam rows dan columns
4. Use AI to generate table data
5. Export sebagai CSV
```

## 💾 Auto-Save & Storage

- **Auto-save**: Dokumen otomatis disimpan setiap 10 detik
- **LocalStorage**: Backup tersimpan di browser
- **Export**: Download file ke komputer
- **Backend**: Option untuk simpan ke server

Lihat statistik dokumen:

- **Character Count** - Jumlah karakter
- **Line Count** - Jumlah baris
- **Cell Count** - Jumlah cell (EXCEL)

## 🎨 Formatting Tips

### DOCX Editor
- Klik pada text untuk edit
- Setiap baris adalah paragraf terpisah
- Multiple paragraphs membuat dokumen lebih terstruktur

### PPTX Editor
- Setiap slide otomatis punya title dan content
- Gunakan AI untuk generate content yang menarik
- Notes section untuk presenter notes

### EXCEL Editor
- Click cell untuk edit
- Tab key untuk move ke cell berikutnya
- Struktur data dengan headers di baris pertama

---

## 💡 Pro Tips

1. **Gunakan Quick Templates** - Lebih cepat dari blank document
2. **Refine dengan AI** - Bisa request revisi/perbaikan
3. **Check Auto-Save** - Lihat notification ketika auto-save
4. **Keyboard Shortcuts**:
   - `Tab` - Next cell (Excel)
   - `Ctrl+S` atau auto-save langsung
   - `Escape` - Close AI panel

5. **Large Documents** - Split into multiple smaller docs untuk performa lebih baik

## ⚠️ Common Issues & Solutions

### "Export tidak berhasil?"
- ✅ Check file size
- ✅ Ensure browser allows downloads
- ✅ Try browser yang berbeda

### "AI generation lambat?"
- ✅ Check internet connection
- ✅ Try shorter prompt
- ✅ Retry dengan simple template

### "Dokumen tidak ter-load?"
- ✅ Refresh browser
- ✅ Check localStorage
- ✅ Try incognito/private mode

### "Content tidak tersimpan?"
- ✅ Auto-save berjalan setiap 10 detik
- ✅ Manual export untuk backup
- ✅ Check browser console untuk error
